# 4-Account Money System
---
## FRAMEWORK SECTIONS:
Breakdown by part:
```markdown
 Income -> Split (
     1. Spend
     2. Save
     3. Invest
     4. Taxes
 )
```
Example:<br>
> $1000<br>
>   -> $300 -> Tax account<br>
>   -> $500 -> Spending account<br>
>   -> $200 -> Savings/Investing accounts
---
## 4-ACCOUNT MONEY SYSTEM BREAKDOWN:
## 1. Spending Account (Checking)
> This is your ***MAIN ACCOUNT***<br>
> 1. Pay bills<br>
> 2. Buy food<br>
> 3. Daily life<br>
## 2. Savings Account (Emergency Fund)
> This is your ***SAFETY NET***<br>
> 1. Car repairs<br>
> 2. Unexpected expenses<br>
## 3. Investing Account
> This is your ***LONG-TERM MONEY***<br>
> 1. Roth IRA<br>
> 2. Brokerage<br>
## 4. Taxing Account (High-Yield Savings)
> This is your ***DON'T TOUCH ACCOUNT***<br>
> 1. Set aside ~25-30% of income<br>
> 2. Earn interest<br>
> 3. Used ONLY for taxes<br>
